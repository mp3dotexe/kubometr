# Kubometr

AI-консультант строительного магазина, разработанный на Go.

Kubometr — Telegram-бот для строительного магазина, использующий Google Gemini для помощи в подборе строительных материалов и консультаций по вопросам ремонта.

Проект развивается как backend-приложение с постепенным добавлением PostgreSQL, Docker и каталога товаров.

## Возможности

- Консультации через Telegram
- Интеграция с Google Gemini
- Ограничение количества AI-запросов
- Ограничение длины сообщений
- Таймауты запросов к AI
- Управление состоянием пользователя
- Разделение транспортного слоя и бизнес-логики

## Технологии

- Go
- Telegram Bot API
- Google Gemini API
- slog
- Docker *(в разработке)*
- PostgreSQL *(в разработке)*

## Архитектура

```text
cmd/
└── bot/

internal/
├── ai/
├── app/
├── config/
├── consultation/
├── logger/
├── state/
└── telegram/
```

### Основные компоненты

| Пакет | Назначение |
|--------|------------|
| `telegram` | Работа с Telegram Bot API |
| `consultation` | Бизнес-логика консультаций |
| `ai` | Клиент Google Gemini |
| `state` | Управление состоянием пользователей |
| `config` | Загрузка конфигурации |
| `logger` | Настройка логирования |
| `app` | Сборка зависимостей и запуск приложения |

## Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/mp3dotexe/kubometr.git
cd kubometr
```

### 2. Создать файл `.env`

Используйте `.env.example` как шаблон.

```env
BOT_TOKEN=
GEMINI_API_KEY=
GEMINI_MODEL=gemini-2.5-flash

AI_TIMEOUT=30s
AI_RATE_LIMIT=5s
MAX_PROMPT_LENGTH=2000
MAX_CONCURRENT_AI=5
```

### 3. Запустить приложение

```bash
go run ./cmd/bot
```
